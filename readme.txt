Readme

Design specification:
The sequence diagram and project abstract is available in the design folder of the parent folder called system-monitoring.

Requirement:
1 – java installed
2 – maven installed.

Howto:
1 – run the maven build 
“mvn clean install”
2 – go to target folder and run application using the below command
“java -jar system-monitoring-jar-with-dependencies.jar”
